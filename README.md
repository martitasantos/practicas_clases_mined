## 🚀 ¿Cómo usar este repositorio?

### 1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/bootcamp-mineducyt.git
cd bootcamp-mineducyt
```

### 2. Navegar a la práctica de tu interés
```bash
cd html-css/practica-01-estructura-basica
```

### 3. Abrir el proyecto
```bash
# Con VS Code
code .

# O simplemente abre el index.html en tu navegador
```

## 👩‍🏫 Para los Docentes

Estimados docentes, este repositorio está diseñado para facilitar la revisión de avances. Cada práctica incluye:

- 📋 **Instrucciones** en su propio `README.md`
- 💡 **Comentarios en el código** explicando cada sección
- ✅ **Checklist de requisitos** cumplidos

Si encuentran algo que mejorar, pueden abrir un **Issue** o dejar comentarios directamente en el commit correspondiente.

---

## 🤝 Cómo Contribuir

1. 🍴 Haz un **fork** del repositorio
2. 🌿 Crea una rama: `git checkout -b practica/nombre-del-tema`
3. 💾 Realiza tus cambios y haz commit: `git commit -m "Add: práctica de flexbox"`
4. 📤 Sube tus cambios: `git push origin practica/nombre-del-tema`
5. 🔄 Abre un **Pull Request**

---

## 📏 Convenciones de Commits

Usamos prefijos para mantener el historial ordenado:

```
✨ feat:    Nueva práctica o funcionalidad
🐛 fix:     Corrección de errores
📝 docs:    Cambios en documentación
💄 style:   Cambios de estilos/formato
♻️  refactor: Mejora de código existente
```
